
# Requisitos do Sistema

## 1. Requisitos Funcionais (RF)

### RF01 — Login do usuário
O sistema deve permitir que o usuário realize login com e-mail e senha.

### RF02 — Recuperar senha
O sistema deve permitir que o usuário solicite recuperação de senha por e-mail.

### RF03 — Atualização de perfil
O usuário pode alterar nome, foto e dados pessoais.

### RF04 — Cadastro de novos usuários
O sistema deve permitir que visitantes criem uma conta fornecendo nome, e-mail e senha.

### RF05 — Gerenciamento de permissões (Níveis de Acesso)
O sistema deve diferenciar as funcionalidades acessíveis entre usuários comuns e administradores.

### RF06 — Notificações do sistema
O sistema deve enviar notificações ao usuário sobre eventos importantes (ex: confirmação de cadastro ou alertas de segurança).

### F07 — Busca e filtragem
O sistema deve permitir que o usuário realize buscas por palavras-chave e aplique filtros em listas de dados.

### RF08 — Logout do usuário
O sistema deve permitir que o usuário encerre sua sessão com segurança, invalidando o token de acesso atual.
